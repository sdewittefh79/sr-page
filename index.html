Scrap Reduction Challenge


Description

The target audience for this 10-minute course is new semiconductor specialists who will be working in a cleanroom environment. It covers basic procedures on proper gowning and handling of semiconductor components.

Tools Used for the Build

The course was authored in Articulate Storyline 360. All videos, icons, and illustrations are from the Articulate Content Library. Stock images were downloaded from Articulate Content Library or Pixabay. Adobe Photoshop was used to add a radial blur effect to the spinning game wheel button. The background music was downloaded as an MP3 file from eLearning Brothers. Audacity was used to create and edit the audio narration. It was also used to create the game wheel spin clicking sound.
